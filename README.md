# React Boilerplate  App

Basic React Boilerplate

## Get Started

Install the node libraries
```
npm install
```

Run the app on "localhost:3000"
```
node server.js
```

## Development tools:
* Main: react
* react-dom
* react-router
* For compiling: webpack, babel
* Bootstrap: foundation-site
* Styles: node-sass
* Test Suite: expect, karma, mocha
